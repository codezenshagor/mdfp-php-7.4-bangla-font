# Bangla PDF Generator with mPDF 7 for PHP 7.4

Generate professional PDFs in **Bangla and English** using **mPDF 7** in **Core PHP (PHP 7.4)**.  
Supports **custom Bangla fonts**, **watermarks**, and **HTML/CSS styled invoices**.

---

## Features

- Supports **Bangla & English text**.
- Works with **PHP 7.4**.
- Uses **mPDF 7.x old syntax** (`new mPDF()`).
- Add **custom fonts** like SolaimanLipi.
- Supports **image & text watermarks**.
- Design **invoices, reports, or letters** using HTML/CSS.
